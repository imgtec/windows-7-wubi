Windows 10 支持说明

1. 安装方法与 Windows 7 相同。

2. 添加繁体中文大易输入法
   a) 添加语言 “繁体中文”，并选择 “中文（繁体，香港特别行政区）”
   b) 添加键盘，选择 “繁体中文大易（版本 6.0）”

3. 功能与变更
   a) 临时切换中英文  Shift -> Ctrl+,
   b) 五笔查询键      z     -> ?
   c) Ctrl+SPACE 无功能。

4. 切换输入法
   Windows 10 可以使用 WIN+SPACE 切换输入法。
   Ctrl + WIN + SPACE：在英文输入法与当前输入法之间来回换切，相当于早期版本的 Ctrl + SPACE。

5. 关于码表
   Windows 10 里的词组不会更新，更新的词组可以从 Windows 7 的码表同步。

2017.7.29

- - - - -
为什么 Windows 10 自带五笔输入法，还需要维护这个输入法。
1. Windows 10 用于简体中文的新输入法引擎的“吃键”问题。
   吃键，这并不是一个专业术语，是我自己发明的一个词。
   现象为，当输入者打字很快时，系统会对一些输入略过处理。
   在系统卡顿时，表现更为明显。结果就是，输出了一串未经转
   化的字母。这个现象至少在如下版本的 Windows 10 中存在。

   [x] Microsoft Windows [版本 10.0.17763.107]

2. 输入汉字范围全面，词组丰富导致重码多的现象。一些常用词
   与多个罕用词形成重码，导致常用词不能形成“四码唯一”上屏。
   这个现象同样存在于单字，例如一个单字与词组或罕用字形成
   重码。

3. 维护个人词库需要工具，可能这个会是最主要的一个，不如仅
   使用文本编辑软件来得方便。

4. Shift 键现象。其实这个现象在上一代的输入法引擎中也存在。
   但是 TextTable 可以使用记事本编辑码表文件来修改，不需要
   借助其它软件。

问七五笔输入法在 Windows 10 上存在的缺陷。

[解决] 在微软 Word 中使用这个输入法，输出的字体会自动切换到“明体”。
[解决] 借壳香港繁体输入法 =|-_-|= ，在语言中被归类为繁体，有碍难观。
